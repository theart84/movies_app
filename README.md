# Movies App

## Веб-приложение отображающие топ 250 фильмов.
![movies_app](https://i.ibb.co/t4Q1hNN/movie-app.jpg)



Реализованные фичи:
+ Поиск по названию
+ Пагинация
+ Прелодер
+ Просмотр подробной информации о фильме
+ Удаление фильма
+ Интерактивная подложка
